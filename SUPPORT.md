# Support

## How to file issues and get help

This project uses GitHub Issues to track bugs and feature requests. Please search the existing
issues before filing new ones to avoid duplicates. For new bugs or feature requests, file them as a new Issue.

## Chat with us on the CNCF Slack

Have questions or need help? Join the conversation in the **#hyperlight** channel on the CNCF Slack!

To get started, [join the CNCF Slack](https://www.cncf.io/membership-faq/#how-do-i-join-cncfs-slack) and then head over to the **#hyperlight** channel. We look forward to hearing from you!

## Microsoft Support Policy

Support for **hyperlight-js** is limited to the resources listed above.
